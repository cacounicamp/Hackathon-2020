Setup

— ! — friend_ [friend_@10.3.2.91]

Eai computeires, vamos se preparar pro evento?

19:06 <CACo> O setup tem como objetivo introduzir alguns conceitos que podem te ajudar a fazer uma solução bem legal no hackathon, e antecipar a configuração do seu computador para rodar o que for necessário.

19:06 <CACo> O que eu vou ver no setup?

19:06 <CACo> Nosso setup foi organizado pra te preparar para o desenvolvimento de um projeto web (em outras palavras, um site).

19:06 <CACo> Portanto, preparamos tipo um mini curso voltado para conceitos essenciais de projetos web que são: HTML, CSS e Javascript.

19:06 <CACo> Além disso, preparamos o setup para te explicar sobre VueJS, que é um framework de javascript, que vai te ajudar a organizar o seu projeto e também no desenvolvimento da lógica que o seu site terá =)

19:06 <CACo> Para isso preparamos apresentações que explicam sobre esse framework, e deixamos pronto um projeto base, que você e seu grupo poderão utilizar no evento.

19:06 <CACo> O setup é dividido em 4 pontos:

19:06 <CACo> 1. HTML e CSS - Elementos de uma página

19:06 <CACo> 2. Javascript - Se familiarizar com a linguagem

19:06 <CACo> 3. VueJS - Organização do projeto e funcionalidades da sua página

19:06 <CACo> 4. Github - Versionar seu projeto

19:06 <CACo> Fique tranquilo se você não teve nenhum contato com essas tecnologias

19:06 <CACo> Nosso setup foi montado para atingir todos os níveis de conhecimento, então foque no início (HTML e CSS)

19:06 <CACo> Não se sinta pressionado, vá até onde você conseguir. O aprendizado leva tempo, e não esperamos que todos saibam tudo

19:06 <CACo> Como o hackathon é em grupo, o objetivo é realmente que você aprenda um pouco, troque ideias e principalmente pense na solução pro problema da democracia nesse momento à distância.

19:06 <CACo> Como vai funcionar o setup?

19:06 <CACo> 1. Montamos um cronograma de estudo para você se organizar

19:06 <CACo> 2. Separamos tutoriais que explicam cada um dos conceitos.

19:06 <CACo> 3. Organizamos meets onde iremos falar sobre os conceitos e tirar dúvidas.

19:06 <CACo> Não perca os meets! Veja o cronograma do setup e os tutoriais no Manual.
