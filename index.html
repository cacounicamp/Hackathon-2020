O que eu vou ver no setup? Nosso setup foi organizado pra te preparar para o desenvolvimento de um projeto web (em outras palavras, um site) Portanto, preparamos apresentações voltadas para conceitos essenciais de projetos web que são: HTML, CSS e Javascript. Além disso, preparamos o setup para te explicar sobre VueJS, que é um framework de javascript, que vai te ajudar a organizar o seu projeto e também no desenvolvimento da lógica que o seu site terá =) Para isso preparamos apresentações que explicam sobre esse framework, e deixamos pronto um projeto base, que você e seu grupo poderão utilizar no evento. ** Portanto, recomendamos bastante que você utilize esse framework, pois nos preparamos para isso e com certeza é um conhecimento que acrescenta bastante na sua experiência desenvolvendo. ** O setup é dividido em 4 pontos: 1. HTML e CSS - Elementos de uma página 2. Javascript - Se familiarizar com a linguagem 3. VueJS - Organização do projeto e funcionalidades da sua página 4. Github - Versionar seu projeto Como vai funcionar o setup? Montamos um cronograma de estudo para você, e organizamos uma coleta de referências que explicam os conceitos. Além disso organizamos meets onde iremos aprofundar sobre os 4 conceitos (Html+css, javascript, vueJS e git) e tirar dúvidas. Nossas apresentações (slides) do meet são sobre cada uma das tecnologias, e elas ficarão disponíveis a partir do primeiro dia do setup.
